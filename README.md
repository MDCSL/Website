# MDCSL 明道資安研究社網站

這是一個簡單且專業的靜態網站，專為明道資安研究社打造。

## 檔案列表

- `index.html` - 網站主頁面
- `style.css` - 樣式表
- `script.js` - JavaScript 互動效果
- `Video.gif` - MDCSL Logo

## 使用方式

1. 將所有檔案放在同一個資料夾內
2. 直接開啟 `index.html` 即可在瀏覽器中查看網站
3. 若要部署到網路上，可以使用以下服務：
   - GitHub Pages
   - Netlify
   - Vercel
   - 或任何靜態網站託管服務

## 特色功能

### 視覺效果
- 🎨 科技感的深色主題設計
- ⚡ 閃電 logo 動畫效果
- 🌟 滑鼠跟隨光暈效果
- 📱 完全響應式設計（支援手機、平板、電腦）

- 提問指南連結

## 自訂設定

### 修改顏色主題
在 `style.css` 檔案的 `:root` 區塊修改：
```css
:root {
    --primary-color: #00ff88;    /* 主要顏色 */
    --secondary-color: #0099ff;  /* 次要顏色 */
    --dark-bg: #0a0a0a;         /* 背景顏色 */
    --card-bg: #1a1a1a;         /* 卡片背景 */
}
```

### 修改內容
直接編輯 `index.html` 中的文字內容即可。
